# Doğum Günü İndirim Şablon Varyasyonları (30 Adet — 15 Seçilen, 15 Yedek)
# Tetik: Müşterinin doğum günü (BIRTHDAY_TODAY)
# Davranış: Yılda bir kez, salon başına gönderilir
#
# Değişkenler:
#   {{customer_name}}      — Customer.firstName
#   {{customer_surname}}   — Customer.lastName
#   {{customer_honorific}} — Customer.gender → "Bey" / "Hanım"
#   {{discount_amount}}    — Salon.birthdayDiscountText (örn. "%15", "150 TL", "1 ücretsiz seans")
#   {{validity_period}}    — Salon.birthdayValidityText (örn. "7 gün", "31 Aralık'a kadar")
#
# ⚠️ META KATEGORİSİ: MARKETING (UTILITY DEĞİL!)
#   - Customer.acceptMarketing = true olmalı (send-time check)
#   - Customer "STOP/DURDUR" derse o anda ban → opt-out
#   - UTILITY'den ~4x pahalı per send
#   - Salon birthdayDiscountText veya birthdayValidityText boşsa
#     template GÖNDERİLMEZ (salon promosyon kullanmıyor)
#
# Ton hiyerarşisi:
#   FRIENDLY     — emoji + {{customer_name}}
#   BALANCED     — emoji + {{customer_name}} {{customer_honorific}}
#   PROFESSIONAL — emojisiz + Sayın {{customer_name}} {{customer_surname}}
